# HR Analytics Dashboard

## Project Overview
HR Analytics Dashboard built using Power BI and Python.

## Features
- Total Employees
- Average Salary
- Attrition Count
- Department-wise Analysis
- Gender Distribution
- Employee Details Table

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI

## Dataset
Employee data containing:
- Employee ID
- Department
- Gender
- Salary
- Experience
- Attrition
