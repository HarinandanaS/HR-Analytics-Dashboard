# HR Analytics Dashboard

## 📊 Project Overview

This project is an HR Analytics Dashboard built using Power BI and Python. It helps analyze employee data and provides insights into workforce distribution, salary trends, and employee attrition.

---

## 🚀 Features

- Total Employee Count
- Average Salary Analysis
- Employee Attrition Analysis
- Department-wise Employee Distribution
- Gender Distribution Analysis
- Employee Details Table
- Interactive Dashboard Filters

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Power BI
- CSV Dataset

---

## 📂 Dataset Information

The dataset contains:

- Employee ID
- Department
- Gender
- Salary
- Experience
- Attrition Status

---

## 📈 Dashboard Insights

### Total Employees
Displays the total number of employees in the organization.

### Average Salary
Shows the average salary of all employees.

### Attrition Count
Displays the number of employees who left the organization.

### Department Analysis
Analyzes employee distribution across departments.

### Gender Distribution
Shows male and female employee percentages.

### Employee Details Table
Provides detailed employee information including salary, experience, and attrition status.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/HarinandanaS/HR-Analytics-Dashboard.git
```

2. Navigate to the project folder

```bash
cd HR-Analytics-Dashboard
```

3. Install required libraries

```bash
pip install pandas matplotlib
```

4. Run the analysis script

```bash
python hr_analysis.py
```
